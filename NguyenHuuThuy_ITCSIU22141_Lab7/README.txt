3. - If you re-introduce the problem from step 1 of the previous section, the code will not compile. 
   - Because generics in Java are type-safe. 
   - When declaring a Pair<String, Integer>, the first object will always be a String and the second object will always be an Integer. If you try to store a different object type, the compiler will return an error.


How to compile and run:
